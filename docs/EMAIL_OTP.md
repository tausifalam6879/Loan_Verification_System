# Email OTP Setup

The OTP code is generated by the Spring Boot backend and sent through SMTP email.

By default OTP is disabled so local login/register works without email setup. To send real OTP emails, enable both OTP and mail settings with environment variables.

## Gmail Setup

1. Open your Google account security settings.
2. Enable 2-Step Verification.
3. Create a Gmail App Password.
4. Use that app password as `SMTP_PASSWORD`.

Do not use your normal Gmail password.

## Start Backend With Email OTP

Run these commands in PowerShell from the project root:

```powershell
$env:APP_OTP_ENABLED="true"
$env:APP_MAIL_ENABLED="true"
$env:SMTP_HOST="smtp.gmail.com"
$env:SMTP_PORT="587"
$env:SMTP_USERNAME="your-email@gmail.com"
$env:SMTP_PASSWORD="your-16-character-gmail-app-password"

.\mvnw.cmd spring-boot:run
```

Keep that backend terminal running.

## IntelliJ Run Configuration

If you run the backend from IntelliJ:

1. Open `Run > Edit Configurations`.
2. Select the Spring Boot run configuration.
3. Add these environment variables:

```text
APP_OTP_ENABLED=true
APP_MAIL_ENABLED=true
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=your-email@gmail.com
SMTP_PASSWORD=your-16-character-gmail-app-password
```

4. Save and run the backend again.

## Frontend Flow

Register:

1. Open `http://localhost:3000/#/register`.
2. Enter full name, email, and password.
3. Click `Send OTP`.
4. Check that email inbox or spam folder.
5. Enter the 6-digit OTP.
6. Click `Verify`.
7. Click `Register`.

Login:

1. Open `http://localhost:3000/#/login`.
2. Enter email and password.
3. Click `Send OTP`.
4. Enter the OTP received by email.
5. Click `Verify`.
6. Click `Login`.

## Troubleshooting

### OTP button says email delivery is disabled

The backend was not started with:

```text
APP_OTP_ENABLED=true
APP_MAIL_ENABLED=true
```

Restart backend after setting those variables.

### OTP email is not received

Check:

- Gmail App Password is correct.
- 2-Step Verification is enabled in Gmail.
- `SMTP_USERNAME` is the same Gmail account.
- Inbox and spam folder.
- Backend terminal for the exact error message.

### Login/Register says OTP verification required

OTP is enabled, but the OTP was not verified before pressing login/register. Send OTP, enter the code, click verify, then submit.
